Projects

  • TDR LSTM Based Autoencoders This repository contains a Jupyter notebook and a technical paper for experiments on Tagalog Diacritic Restoration (TDR) using LSTM-based autoencoders. The dataset is based on articles from the Sari-Samot blogs by Filipino NA Virgilio Almario, which has diacritized homographs.
  • TDR Transformer Based Models Partial codes for the thesis project focused on improving Tagalog diacritic restoration using corpus expansion, linguistic analysis, and transformer-based models.
  • FIES Viewer A Django-based web application for exploring and analyzing household economic data from the Family Income and Expenditure Survey (FIES) in the CALABARZON Region, Philippines. This project was developed for the Programming with Databases course in the MDSc program at Ateneo de Manila University.
  • SIMS: Social Impact Mapping Simulation A Java-based two-party election simulation implementing principles of Social Impact Theory (Nowak, Szamrej, & Latané, 1990). This project was developed for the Programming with Databases class in MDSc at Ateneo de Manila University. It showcases a simple GUI, basic Java programming concepts, and simulation of social influence dynamics in a two-party election scenario.
